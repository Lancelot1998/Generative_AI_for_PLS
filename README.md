## Files in .\lyq_data\pythoncode\DeepSCmaster

### `test_Qwen.py`

**Single-scenario brute-force baseline.**
In an Eve scenario generated by Qwen, the script enumerates candidate `(P, k)` pairs and reports the best action and the corresponding performance metrics.

### `test_drl.py`

**Standard PPO training under Qwen-generated scenarios.**
This script uses Qwen to generate Eve scenarios, builds the environment, performs multi-scenario PPO training, and saves the final trained policy.

### `test_drl_nollm.py`

**No-LLM control experiments.**
In addition to Qwen-based scenarios, this script supports randomly generated Eve scenarios for ablation/control comparisons against the LLM-based setting.

### `test_Qwen_teacher.py`

**Teacher-guided PPO training with an LLM teacher.**
Built on Qwen-generated scenarios, this script introduces teacher action suggestions and injects an auxiliary reward to guide PPO learning, aiming to improve search/training efficiency.
